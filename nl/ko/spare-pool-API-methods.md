# 예비 풀 API 메소드
{: #bm-api-spare-pools}

API 메소드를 사용하여 예비 풀을 관리할 수 있습니다. 

{:shortdesc}

| 메소드 |설명|
|------|-----------|
|getDaysInSparePool|서버가 예비 풀에 있었던 일 수를 검색합니다. |
|massSparePool|여러 서버가 예비 풀에 추가되거나 예비 풀에서 제거되도록 허용합니다. |
|sparePool|서버가 예비 풀에 추가되거나 예비 풀에서 제거되도록 허용합니다. |
